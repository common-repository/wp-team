== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use WP Team in your post, page, widget or anywhere you like. How to create WP Team? lets see.


1. Upload the folder wp-team to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
4. Add member going to wp-admin -> WP Team -> Add New Member.
5. Call shortcode to your WordPress editor directly, by using 
`
[wp-team]
`
Call shortcode to your php template file using 
`
<?php  echo do_shortcode(‘[wp-team]'); ?>
`

Now save your page, then visit the newly created page & see something awesome. :)
